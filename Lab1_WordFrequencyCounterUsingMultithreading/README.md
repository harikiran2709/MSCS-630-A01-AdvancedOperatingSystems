# Word Frequency Counter Using Multithreading

## Requirements
- Python 3.x

## How to Run

1. Place your input text file (e.g., `sample.txt`) in this directory.
2. Run the script from the terminal:

    python word_freq_multithread.py <input_file> <num_segments>

   Example:

    python word_freq_multithread_wordsplit.py sample.txt 4

- `<input_file>`: Path to your text file.
- `<num_segments>`: Number of segments/threads to use (positive integer).


# Project Deliverable 4: Integrated Shell

## Instructions

1. Open a terminal in this folder.
2. Run the integrated shell:
   
   python3 shell_integrated.py

3. Log in with a username and password (see auth.py for sample users).
4. Use shell commands as usual. Piping is supported (e.g., ls | grep txt).
5. To access demos from previous deliverables, type:
   
   menu

   and follow the prompts to launch:
   - Basic Shell (Deliverable 1)
   - Scheduling Demo (Deliverable 2)
   - Memory Management & Synchronization Demo (Deliverable 3)

## Security
- User authentication required to access shell.
- File permissions are simulated per user.

## Example Users
- admin / adminpass
- user1 / user1pass
- user2 / user2pass

## Files
- shell_integrated.py: Main shell
- auth.py: User authentication and permissions
